# Job Market Intelligence Platform

A full end-to-end data science project analyzing US and India job markets.

## Live Demo
https://job-market-intelligence-platform-qwrlwbvcejfux5zcktqvrt.streamlit.app

## Features
- Salary Explorer — analyze salaries by experience and role
- Skills Analysis — top in-demand skills + skill gap checker
- India vs Global — compare job markets
- AI Salary Predictor — predict salary using ML

## Datasets
- Data Analyst Jobs (Kaggle)
- DS Salaries (Kaggle)
- Naukri Jobs India (Kaggle)

## Tech Stack
Python, Pandas, Scikit-learn, Plotly, Streamlit, NumPy, Matplotlib, Seaborn

##  Model Performance
- Algorithm: Random Forest Regressor
- R² Score: 0.50
- MAE: $29,648

##  How to Run Locally
pip install -r requirements.txt
streamlit run dashboard/app.py
